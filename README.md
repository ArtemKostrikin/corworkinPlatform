# CoworkinPlatform

Платформа для бронирования рабочих мест в коворкингах.

## Функционал
- Регистрация и авторизация
- Просмотр списка коворкингов
- Поиск и фильтрация по городу и цене
- Бронирование рабочих мест
- Управление личным кабинетом

## Стек
- Nuxt 3
- TypeScript
- Pinia
- Vite

## Запуск

```bash
npm install
npm run dev
